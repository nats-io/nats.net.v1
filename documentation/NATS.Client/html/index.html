NATS .NET client.

Introduction

The NATS .NET Client is part of NATS an open-source, cloud-native messaging system, and is supported by Apcera. This client, written in C#, follows the go client closely, but diverges in places to follow the common design semantics of a .NET API.

Installation

Instructions to build and install the NATS .NET C# client can be found at the NATS .NET C# GitHub page
