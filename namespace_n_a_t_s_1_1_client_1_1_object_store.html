Enumerator
IgnoreDelete 

Do not include deletes or purges in results. Default is to include deletes.

IncludeHistory 

Watch starting at the first entry for all keys. Default is to start at the last per key.

UpdatesOnly 

Watch starting when there are new entries for keys. Default is to start at the last per key.
